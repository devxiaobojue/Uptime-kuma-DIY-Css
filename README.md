

# Uptime Kuma DIY CSS

开源监控系统 Uptime Kuma 的自定义 CSS 样式库。

## 简介

本项目提供了一套精美的自定义 CSS 样式，用于美化 Uptime Kuma 监控系统的状态页面。通过应用这些 CSS 样式，您可以轻松打造独具特色的监控面板展示效果。

## 功能特点

- 🎨 精美的主题样式设计
- 🔧 开箱即用，快速部署
- 📱 响应式布局，适配各种设备
- ✨ 提升状态页面的视觉体验

## 使用方法

1. 复制本仓库中的全部 CSS 数据
2. 登录 Uptime Kuma 管理面板
3. 进入 **状态页面 (Status Page)**
4. 点击 **编辑状态页面 (Edit Status Page)**
5. 在 **自定义 CSS (Custom CSS)** 中删除原有内容
6. 粘贴本项目的 CSS 数据
7. 保存设置

## 适用版本

适用于 Uptime Kuma 开源监控系统的各版本。
